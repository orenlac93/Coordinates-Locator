# Coordinates-Locator

<br />
Python application for finding location on the map by a given coordinates

<br />
<br />

### requiremets

1. Python 3 <br />
2. folium <br />
3. tkinter <br />

<br />
<br />

### How to use

1. run the 'App.py' file <br /> <br />
2. in the application window enter the location coordinates and push 'find location' button
